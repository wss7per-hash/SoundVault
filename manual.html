软件介绍

SoundVault 是什么？

它是为音效设计师、游戏/影视开发者、播客与视频创作者打造的「本地音效素材库」。你导入成千上万的音频片段，它帮你做三件事：听懂（AI 语义分析）、归类（标签与智能文件夹）、复用（搜索、收藏、跨设备迁移）。所有数据都在你自己的电脑上，API Key 只存在本地。

为什么不用网盘 + 文件夹？ 当素材超过几百条，「按文件名猜内容」就会崩溃。SoundVault 让每段音频自带结构化描述与标签——你可以用「战斗、紧张、金属」这样的语义组合来检索，而不是记住每段文件的名字。
全新品牌标识 本次版本启用全新图标——金属质感的 SV 立方体，融合音箱、声波与音符元素，呼应「声音保险库（Sound Vault）」的定位。

导入音频

① 左侧栏顶部「导入」图标 ⤓  或  中间空状态的导入入口

对话框步骤config（配置）→ scanning（扫描中）→ preview（扫描结果）→ importing（导入中）→ done（完成）
目标文件夹点「点击选择文件夹」选一个目录；可点「+ 添加更多文件夹」一次选多个
快速选项递归子目录（默认勾选）· 跳过隐藏文件（默认勾选）· 包含视频文件
高级过滤 ▸展开后：文件名包含（如 sword,impact）、文件名排除（如 temp,draft）、最小/最大大小(KB)
自动标记按文件夹结构自动标记标签（默认勾选，按目录层级生成标签）

  • 音频本体复制进本地库目录，file_path 指向本机位置——这也是它能离线工作的原因。
  • 导入完成后音效出现在中间主区；若文件本体丢失，会标记为「缺失」但仍保留元数据。

AI 语义分析

右侧详情面板  ·  顶部醒目「AI 分析」按钮

点「AI 分析」按钮，AI 会输出：形象描述、使用场景、关联关键词、拟声词，以及一组语义标签。

按钮状态未分析＝紫色「AI 分析」；分析中＝「分析中…（点击取消）」可随时取消；已分析＝「重新 AI 分析」
输出位置描述进「AI 描述」区、场景进「适用场景」chip、情绪进「情绪」、标签进「标签」区并归入左侧八大分类
首次配置没配 Key 时点分析→ 弹友好引导提示并自动弹出模型配置面板（见 Step 9 细节）
批量分析右侧面板可对每个音效单独分析；「导入时勾选自动分析」可批量完成

智能文件夹

智能文件夹不是物理文件夹，而是一组自动归类条件。建好后符合条件的音效自动出现其中。

规则弹窗520px 弹窗：顶部名称输入框 + 条件区 + 底部「当前匹配 N 个音效」实时计数 + 「保存并应用」
11 个字段文件名 / AI描述 / 情绪 / 适用场景 / 标签分类 / 文件格式 / 时长(ms) / 质量评分 / 已收藏 / 文件缺失 / 分析状态 / 导入时间(天)
操作符包含 / 不含 / 等于 / 开头是 / 大于 / 小于 / 是（布尔字段用「是/否」，分析状态用「已分析/未分析」）
多条件每组内可「+ 添加条件」；「AND·全部满足」按钮切换为「OR·任一满足」；「+ 添加条件组」增加一组（组间默认 AND）
列表项智能 Tab 内点击即应用筛选；hover 显示编辑 / 删除
  • 支持 11 个字段，多条件可分组，组内「与 / 或」自由组合。
  • 编辑时底部实时显示「当前匹配 N 个音效」，调条件即刻看到结果。
  • 保存后，符合条件的音效会自动出现在该智能文件夹里，新增素材也会自动归位。

常见问题

你可能想问

我的音频会被上传到云端吗？
不会。SoundVault 完全本地运行，音频文件只存在你电脑上。只有当你配置并启用了 AI 分析时，才会把文本描述发送给你所选的模型服务商（Key 仅存本机）；音频本体不上传。
点「AI 分析」提示要配置 API Key，怎么办？
这是正常的首次引导，不是报错。SoundVault 会弹出一条友好的提示，并自动打开模型配置面板。你在面板里选好服务商（如 DeepSeek / OpenAI）、粘贴 API Key 保存后，关掉面板重新点「AI 分析」即可。Key 只存在你本机，不会外泄。
为什么某个音效在收藏夹里看不到？
请确认已成功「加入收藏夹」并刷新。加入操作需要文件确实写入数据库；若当时提示成功却看不到，重新操作一次即可。星标（⭐）与收藏夹是两套独立机制——星标的音效会出现在收藏夹栏顶部的「⭐ 收藏」虚拟项里。
导出后文件名怎么全变成随机字符串了？
早期版本确实存在该问题，现已修复——导出会保留原始文件名，仅在同名冲突时自动追加 _2 _3 序号。
换了电脑，导入后还能播放吗？
可以。导入时会把音频文件复制到新电脑的库目录并更新路径，所以迁移后照常播放，不依赖旧机器的文件路径。
导出能不能看进度 / 中途取消？
可以。导出时弹出进度面板，显示进度条、已处理数量、当前文件名、实时用时，并有「取消导出」按钮；取消会自动删除未完成的半成品包。完成后还会弹一条带总用时的提示，可一键打开导出文件夹。
导入时文件太多，能只导一部分吗？
可以。扫描结果页的文件列表可勾选，或用「高级过滤」按文件名包含/排除、大小范围预先筛掉；也可以先建好收藏夹，之后用「按收藏夹导出」只带走那一组。
启动后为什么要先看一段动画？
那是品牌启动动画（粒子 + 声波 + 均衡器音柱），最短展示约 1.4 秒、最长 4.5 秒兜底，期间在做数据库与界面初始化。若你用的是单文件便携版 exe，双击后还会先有一次性的自解压（约 1–2 秒黑屏），这是打包形态固有的，安装版/解压目录版无此等待。
