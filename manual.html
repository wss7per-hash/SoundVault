软件介绍

SoundVault 是什么？

它是为音效设计师、游戏/影视开发者、播客与视频创作者打造的「本地音效素材库」。你导入成千上万的音频片段，它帮你做三件事：听懂（AI 语义分析）、归类（标签与智能文件夹）、复用（搜索、收藏、跨设备迁移）。所有数据都在你自己的电脑上，API Key 只存在本地。

为什么不用网盘 + 文件夹？ 当素材超过几百条，「按文件名猜内容」就会崩溃。SoundVault 让每段音频自带结构化描述与标签——你可以用「战斗、紧张、金属」这样的语义组合来检索，而不是记住每段文件的名字。

使用手册

八步上手 SoundVault

从空白到井井有条，跟着走一遍即可。所有操作都在主界面的工具栏与左侧栏完成。

1

导入音频

点击窗口右上角或工具栏的「扫描 / 导入」入口，选择单个文件或整个文件夹。SoundVault 会递归读取常见格式（mp3 / wav / flac / ogg / m4a / aiff 等）。

  • 导入时自动计算文件哈希，重复文件不会重复入库。
  • 音频本体复制进本地库目录，file_path 指向本机位置——这也是它能离线工作的原因。
导入完成后，音效会显示在右侧主区域。若文件本体丢失，会标记为「缺失」但仍保留元数据。
2

播放与管理

点击任意音效卡片，详情面板展示波形、播放 / 暂停、音量滑块与可拖动的进度条。

  • 工具栏可切换 网格视图 与 列表视图。
  • 右键音效可：打开文件位置、复制、重命名、收藏、AI 分析、删除等。
  • 按 文件名 / 时长 / 大小 / 导入时间 排序，支持升 / 降序与格式过滤。
3

AI 语义分析

在详情面板或右键菜单点「AI 分析」。AI 会输出：形象描述、使用场景、关联关键词、拟声词，以及一组语义标签。

  • 首次使用前，在「设置 / 模型配置」里选择服务商并填入 API Key（仅存本机）。
  • 支持并发批量分析，并可在进行中随时取消。
  • 分析结果会自动归入八大分类父节点，形成「分类 → 子标签」层级。
小技巧：标签是检索的核心。分析得越充分，后面用智能文件夹与搜索时能筛得越准。
4

标签体系与筛选

左侧「标签」栏呈现八大分类树。点击任意标签，右侧立即过滤出带该标签的音效。

  • 右键音效可手动添加标签，手动标签与 AI 标签统一管理。
  • 顶部搜索框支持按文件名、场景、情绪关键词实时检索。
5

收藏与收藏夹

两种「收藏」方式并存，各司其职：

  • ⭐ 收藏（星标）：详情面板一键点亮，是全局「我喜欢」的扁平清单；状态栏实时显示「X 个收藏」。
  • 📁 收藏夹：左侧栏的命名分组。右键音效 → 「加入收藏夹」→ 选择已有文件夹，或当场输入名称新建并加入。
  • 收藏夹顶部还内置「⭐ 收藏」虚拟项，星标过的音效也会在这里出现（但不归属于任何具体文件夹）。
6

智能文件夹

左侧点「智能」→ 点 + 新建规则。它不是物理文件夹，而是一组自动归类条件。

  • 支持 11 个字段：文件名、描述、情绪、适用场景、标签、格式、时长、质量评分、是否收藏、是否缺失、分析时间、导入时间。
  • 多条件可分组，组内「与 / 或」自由组合。
  • 编辑时底部实时显示「当前匹配 N 个音效」，调条件即刻看到结果。
  • 保存后，符合条件的音效会自动出现在该智能文件夹里，新增素材也会自动归位。
7

资源库导出 / 导入（跨电脑迁移）

工具栏的 导出库 / 导入库 按钮，生成类似 Eagle 的便携式资源包。

  • 导出：点击后先选范围——全部 / 选中项 / 指定收藏夹，再选保存位置。包内含 library.json（元数据）+ audios/（真实音频，保留原始文件名）+ 说明文档。
  • 导入：选择导出包目录，自动合并进当前库——按哈希去重、标签按名复用、收藏夹按名合并，已有数据不丢失。
  • 导出的音频会复制到本机库目录并更新路径，因此换一台电脑也能正常播放。
注意：导入采用「合并」而非「覆盖」，最安全。若你想完全替换当前库，请先备份数据库文件。
8

界面与效率

  • 工具栏右侧可调节界面字号（12–20px），适配不同屏幕与视力。
  • 无边框窗口，右上角提供最小化 / 最大化 / 关闭。
  • 删除的音效先进入回收站，误删可恢复。
常见问题

你可能想问

我的音频会被上传到云端吗？
不会。SoundVault 完全本地运行，音频文件只存在你电脑上。只有当你配置并启用了 AI 分析时，才会把文本描述发送给你所选的模型服务商（Key 仅存本机）；音频本体不上传。
为什么某个音效在收藏夹里看不到？
请确认已成功「加入收藏夹」并刷新。加入操作需要文件确实写入数据库；若当时提示成功却看不到，重新操作一次即可。星标（⭐）与收藏夹是两套独立机制——星标的音效会出现在收藏夹栏顶部的「⭐ 收藏」虚拟项里。
导出后文件名怎么全变成随机字符串了？
早期版本确实存在该问题，现已修复——导出会保留原始文件名，仅在同名冲突时自动追加 _2 _3 序号。
换了电脑，导入后还能播放吗？
可以。导入时会把音频文件复制到新电脑的库目录并更新路径，所以迁移后照常播放，不依赖旧机器的文件路径。
未来计划

路线图（规划中）

以下方向已在规划中，按优先级推进：

  • 🔊 提示词生成音效（text-to-SFX）——接入 ElevenLabs SFX / Stable Audio，按文字描述直接生成音效，优先级最高。
  • 🔁 首尾无缝循环——纯 ffmpeg 交叉淡化，无需 AI，性价比最高。
  • 🧬 生成类似音频——用本库 AI 分析结果（场景 / 拟声词）作 prompt 近似实现。
  • ⏱️ 延长当前音频——DSP 时间拉伸或 Stable Audio inpainting。
生成类功能需接入外部音频生成 API 或本地部署模型（Stable Audio / AudioCraft），可能按次计费或需 GPU。Suno / Udio 偏音乐、API 受限，暂不纳入。
