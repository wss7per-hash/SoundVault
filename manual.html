软件介绍

SoundVault 是什么？

它是为音效设计师、游戏/影视开发者、播客与视频创作者打造的「本地音效素材库」。你导入成千上万的音频片段，它帮你做三件事：听懂（AI 语义分析）、归类（标签与智能文件夹）、复用（搜索、收藏、跨设备迁移）。所有数据都在你自己的电脑上，API Key 只存在本地。

为什么不用网盘 + 文件夹？ 当素材超过几百条，「按文件名猜内容」就会崩溃。SoundVault 让每段音频自带结构化描述与标签——你可以用「战斗、紧张、金属」这样的语义组合来检索，而不是记住每段文件的名字。
全新品牌标识 本次版本启用全新图标——金属质感的 SV 立方体，融合音箱、声波与音符元素，呼应「声音保险库（Sound Vault）」的定位。

① 左侧栏 — SoundVault logo；四个导航 Tab（标签 · 收藏夹 · 智能 · 回收站） ② 标签树 — 八大分类可展开，点击即筛选中间网格 ③ 工具栏 — 搜索框（可拖拽右边缘调宽）· 全局搜索快捷键提示 ·「库管理 ▾」下拉（导入 / 导出 / 清理） ④ 右侧工具组 — 筛选排序 · 网格/列表切换 · 网格密度 · 设置 · 库洞察 · AI 生成 · 窗口控制 ⑤ 音效卡片网格 — 波形图 + 文件名 + AI 描述 + 时长/格式/大小，悬停预览 ⑥ 右侧详情面板 — 音频播放器、AI 描述、适用场景、情绪、标签、质量评分、技术信息、操作按钮 ⑦ 底部状态栏 — 总量 / 收藏数 / 文件总大小 / 选中数量
核心特性

它能为你做什么

从导入到复用，覆盖音效管理的完整链路。下面的「八步手册」会逐条教你怎么操作。

🧠

AI 语义分析

一键让 AI 化身「音效库标注员」，为每段音频生成形象描述、使用场景、关联关键词、拟声词以及语义标签——全程不用任何音频工程术语，人人能懂。

🗂️

八大分类标签体系

人声 / 动物 / 环境氛围 / 动作物品 / UI交互 / 乐器音乐 / 机械科技 / 特殊效果。AI 分析结果自动归入对应父分类，左侧标签树点击即筛选。

🎚️

波形播放器

右侧详情面板内置音频波形与进度条，支持播放/暂停、±5 秒快退快进、点击波形任意位置定位。私有 sv:// 协议流式加载，大文件也能秒开、可精准 seek。

⭐

收藏与收藏夹

单条音效一键「收藏」（星标）；也可建命名收藏夹做手动分组，右键任意音效即可加入，或当场新建文件夹。

🔁

首尾无缝循环

详情面板底部「首尾无缝循环」：选好交叉淡变时长（默认 30ms，10–500ms），一键用 ffmpeg 把尾音交叉淡入到开头，在原文件同目录生成 *_loop.wav。生成的是「天生无缝」的文件，在任意播放器 / 游戏引擎里循环都不咔哒，且不覆盖原文件。

🪄

智能文件夹

不是文件夹，是自动归类规则。按质量评分、标签、是否收藏、导入时间等 11 个字段设条件，实时预览匹配数量，新增音效自动归位。

📦

资源库导出 / 导入

像 Eagle 一样生成便携式资源包：真实音频 + AI 分析 + 标签 + 收藏夹全部带走，可在另一台电脑一键合并迁入。

🔍

语义检索与筛选

按文件名、场景、情绪关键词搜索；支持按导入时间 / 文件名 / 时长 / 大小排序，按格式过滤，网格与列表双视图切换。

♻️

回收站与隐私优先

删除的音效先进回收站，可恢复。核心管理离线运行，AI 服务商可自定义（DeepSeek / OpenAI / 通义千问 / Kimi / 豆包 / 硅基流动 / Claude / Gemini / Azure / Ollama / TokenDance / 自定义网关），Key 仅存本地。

✨

AI 生成音效（云端）

用文字描述直接生成音效并自动入库。双服务商：Fal.ai（Stable Audio 2.5，最长 190s）/ ElevenLabs（SFX V2，音质最佳）。新用户有免费试用金，按次计费，生成前确认花费。

使用手册

上手 SoundVault

从空白到井井有条，跟着走一遍即可。每一步都标注了操作路径（在哪点）和界面细节（会看到什么、字段叫什么）。

导入音频

① 左侧栏顶部「导入」图标 ⤓  或  中间空状态的导入入口

点击后弹出「扫描音效文件夹」对话框，分五步走完导入。

对话框步骤config（配置）→ scanning（扫描中）→ preview（扫描结果）→ importing（导入中）→ done（完成）
目标文件夹点「点击选择文件夹」选一个目录；可点「+ 添加更多文件夹」一次选多个
快速选项递归子目录（默认勾选）· 跳过隐藏文件（默认勾选）· 包含视频文件
高级过滤 ▸展开后：文件名包含（如 sword,impact）、文件名排除（如 temp,draft）、最小/最大大小(KB)
自动标记按文件夹结构自动标记标签（默认勾选，按目录层级生成标签）
开始选好文件夹后点右下角紫色「开始扫描」按钮
结果页四张统计卡（总计 / 新文件 / 已导入 / 总大小）+ 格式分布条 + 可勾选文件列表（全选/取消）+ 可选「导入后自动 AI 分析」→ 点「导入选中 (N)」
  • 导入时自动计算文件哈希，重复文件不会重复入库（已在库里的会标在「已导入」计数里跳过）。
  • 音频本体复制进本地库目录，file_path 指向本机位置——这也是它能离线工作的原因。
  • 导入完成后音效出现在中间主区；若文件本体丢失，会标记为「缺失」但仍保留元数据。

右键卡片弹出菜单：打开文件位置 / 重命名 / 加入收藏夹 / AI 分析 / 删除

AI 语义分析

右侧详情面板  ·  顶部醒目「AI 分析」按钮

点「AI 分析」按钮，AI 会输出：形象描述、使用场景、关联关键词、拟声词，以及一组语义标签。

按钮状态未分析＝紫色「AI 分析」；分析中＝「分析中…（点击取消）」可随时取消；已分析＝「重新 AI 分析」
输出位置描述进「AI 描述」区、场景进「适用场景」chip、情绪进「情绪」、标签进「标签」区并归入左侧八大分类
首次配置没配 Key 时点分析→ 弹友好引导提示并自动弹出模型配置面板（见 Step 9 细节）
批量分析右侧面板可对每个音效单独分析；「导入时勾选自动分析」可批量完成
  • 首次使用前，在「设置 / 模型配置」里选择服务商并填入 API Key（仅存本机）。
  • 没配 Key 也不慌：点分析不会弹出看不懂的报错，而是显示友好引导提示并自动弹出模型配置面板，按提示选服务商、填 Key 即可。
  • 分析结果自动归入八大分类父节点，形成「分类 → 子标签」层级。
小技巧：标签是检索的核心。分析得越充分，后面用智能文件夹与搜索时能筛得越准。

  • 右键音效可手动添加标签，手动标签与 AI 标签统一管理。

收藏与收藏夹

左侧栏  ·  第二个 Tab「收藏夹」  或   右键卡片 → 加入收藏夹

两种「收藏」方式并存，各司其职：

⭐ 收藏（星标）右侧面板「收藏」按钮一键点亮，是全局「我喜欢」扁平清单；左侧收藏夹栏顶部有虚拟「收藏」项汇总所有星标音效
📁 收藏夹左侧栏的命名分组。点「+」新建（内联输入名称→ 回车创建）；列表项 hover 显示 ✎重命名 / 🗑删除
加入流程右键卡片 →「加入收藏夹」→ 飞出子菜单：选已有文件夹，或「新建收藏夹」输入名称→ 创建并加入
点击筛选点某个收藏夹 → 中间主区只显示该夹内音效（自动切到收藏夹 Tab）
  • ⭐ 收藏（星标）：详情面板一键点亮，是全局「我喜欢」的扁平清单；状态栏实时显示「X 个收藏」。
  • 📁 收藏夹：左侧栏的命名分组。右键音效 → 「加入收藏夹」→ 选择已有文件夹，或当场输入名称新建并加入。
  • 收藏夹顶部还内置「⭐ 收藏」虚拟项，星标过的音效也会在这里出现（但不归属于任何具体文件夹）。
6

智能文件夹

左侧栏  ·  第三个 Tab「智能」  →   点「+」新建规则

智能文件夹不是物理文件夹，而是一组自动归类条件。建好后符合条件的音效自动出现其中。

✨ 一键智能分类：智能 Tab 顶部新增「一键智能分类」面板，按维度自动生成智能文件夹，无需手动写规则。
8 个维度按适用场景 / 按情绪 / 按 AI 标签 / 按文件名（内置中英文关键词词典）/ 按格式 / 按导入时间（今天 · 近7天 · 近30天 · 近90天 · 更早）/ 按时长（短<1秒 · 中1-5秒 · 长>5秒）/ 按质量评分（5★ · 4★ · 3★及以下）
🧠 语义聚类开放的三个维度（场景 / 情绪 / AI标签）不会「一个值一个夹」——而是先把零散值按主题词典归并成少量「主题文件夹」（如「战斗与动作」「环境与自然」「UI与交互」「紧张刺激」…），每个主题夹内含全部相关音效。避免几十个碎夹。
数量可控面板下方有两个可调参数：上限（每个维度最多生成几个文件夹，默认 8）与忽略 < N 个素材（低于该数量的噪声分类直接跳过，默认 2）。超出的主题会合并进「其他」。
一键整理全部面板底部「⚡ 一键整理全部维度」按钮，按当前上限/阈值一次性跑完 8 个维度，省去逐个点击。
幂等安全重复点击不会刷出重复文件夹；某维度没有可归类的音效时会提示「没有可用于该维度的音效」。
即用生成后同名智能文件夹直接出现在列表，点一下即筛选对应音效，后续新增素材也会自动归位。
规则弹窗520px 弹窗：顶部名称输入框 + 条件区 + 底部「当前匹配 N 个音效」实时计数 + 「保存并应用」
11 个字段文件名 / AI描述 / 情绪 / 适用场景 / 标签分类 / 文件格式 / 时长(ms) / 质量评分 / 已收藏 / 文件缺失 / 分析状态 / 导入时间(天)
操作符包含 / 不含 / 等于 / 开头是 / 大于 / 小于 / 是（布尔字段用「是/否」，分析状态用「已分析/未分析」）
多条件每组内可「+ 添加条件」；「AND·全部满足」按钮切换为「OR·任一满足」；「+ 添加条件组」增加一组（组间默认 AND）
列表项智能 Tab 内点击即应用筛选；hover 显示编辑 / 删除
  • 支持 11 个字段，多条件可分组，组内「与 / 或」自由组合。
  • 编辑时底部实时显示「当前匹配 N 个音效」，调条件即刻看到结果。
  • 保存后，符合条件的音效会自动出现在该智能文件夹里，新增素材也会自动归位。

资源库导出 / 导入（跨电脑迁移）

工具栏「库管理 ▾」下拉  ·  导入音效库 / 导出音效库

工具栏的「导出 / 导入」按钮，生成类似 Eagle 的便携式资源包。

导出范围点「导出音效库」→ 弹面板：全部导出（整库）/ 自定义导出（手动勾选 · 按智能文件夹 · 按标签 · 按收藏夹，可单选或组合）
导出进度全屏面板：进度条 +「已处理 X / 总数」+ 实时用时 + 当前正在复制的文件名 + 已复制/缺失计数 +「取消导出」按钮（取消自动清理半成品）
完成提示弹出含总用时的成功 toast（复制 N 个音频 / 缺失 M 个）+「打开文件夹」按钮一键定位
导入点「⤒ 导入」→ 选导出包目录→ 全屏「正在导入资源库…」遮罩→ 自动合并进当前库
文件名导出保留原始文件名，仅在同名冲突时自动追加 _2 _3 序号
  • 导出：选范围 → 选保存位置 → 看实时进度 → 完成弹含用时的提示。
  • 导入：选择导出包目录，自动合并进当前库——按哈希去重、标签按名复用、收藏夹按名合并，已有数据不丢失。
  • 导出的音频会复制到本机库目录并更新路径，因此换一台电脑也能正常播放。
注意：导入采用「合并」而非「覆盖」，最安全。若你想完全替换当前库，请先备份数据库文件。

拟声词（多语种 + 拼音）

选中音效  ·  详情面板「信息」Tab 的「拟声词」卡片   或   「库洞察」面板的「拟声词云」

AI 分析音效时，除原有的中文拟声词外，现在还会一并生成日文拟声词、英文拟声词和中文拼音。无论你用中文"哗啦啦"、日文"ザバザバ"还是英文"splash"描述同一个声音，都能被检索到——特别适合跨语言协作与配音朗读前的快速认读。

  • 详情面板·拟声词卡片：选中音效后在「信息」Tab 可见。每张卡显示中文拟声词（大字）、拼音（如 huā lā lā），以及日/英对照，一目了然。
  • 库洞察·拟声词云：顶部「库洞察」进入，新增全库拟声词聚合词云，字号越大代表该拟声词在库中越多；点击任一词即跳回库并按它搜索。

查找并清除重复文件

顶部「库洞察」 ·  滚动到底部「重复文件」区块

导入时系统已按文件内容（sha256 指纹）自动去重，因此正常导入不会产生重复。但早期版本导入、或手工放进库目录的音频，仍可能在库里形成重复组。本功能让你主动扫描整个音效库、找出所有内容完全相同的文件，并安全地清理多余副本。

  • 精确查重：按文件二进制 hash 比对，只有内容 100% 相同才判定为重复——绝不会误删只是"听起来像"的不同音效。
  • 分组展示：每组列出所有相同文件（文件名、大小、导入时间），并默认保留最早导入的那一个作为原件。
  • 灵活保留：点选组内任意文件可切换"保留"项，其余标记为待删；支持"删除本组其余 N 个"，也支持"一键清理全部重复"。
  • 安全清除：被清理的副本会移入系统回收站（与右键删除一致），可随时从回收站恢复，不会永久丢失。
典型场景：从多个来源批量导入素材后，点一次「查找重复文件」，把几十个重复副本一键清掉，库瞬间清爽、还能腾出磁盘空间。

常见问题

你可能想问

我的音频会被上传到云端吗？
你现有音库里的文件绝不会上传——它们只存在你电脑上，AI 语义分析也只把文本描述发给你所选的模型服务商（Key 仅存本机）。唯一会联网的是可选的「AI 生成音效」功能：它把一句文字描述发给云端模型，再把云端新生成的音频下载进你的库——过程中不会读取或外传你已有的任何音效文件。
AI 生成音效要花钱吗？
两项云端服务商都按次计费：Fal.ai 约 $0.20/次（最长 190s），ElevenLabs 约 $0.10/次（最长 20s）。Fal.ai 新用户送 ~$1–5 试用金，可点生成面板的「额度」按钮查余额，用完自动转付费。生成前一定会弹「约消耗 $X」确认框，不会静默扣费；累计次数与花费只作本地估算、以服务商实际扣费为准。
点「AI 分析」提示要配置 API Key，怎么办？
这是正常的首次引导，不是报错。SoundVault 会弹出一条友好的提示，并自动打开模型配置面板。你在面板里选好服务商（如 DeepSeek / OpenAI）、粘贴 API Key 保存后，关掉面板重新点「AI 分析」即可。Key 只存在你本机，不会外泄。
为什么某个音效在收藏夹里看不到？
请确认已成功「加入收藏夹」并刷新。加入操作需要文件确实写入数据库；若当时提示成功却看不到，重新操作一次即可。星标（⭐）与收藏夹是两套独立机制——星标的音效会出现在收藏夹栏顶部的「⭐ 收藏」虚拟项里。
导出后文件名怎么全变成随机字符串了？
早期版本确实存在该问题，现已修复——导出会保留原始文件名，仅在同名冲突时自动追加 _2 _3 序号。
换了电脑，导入后还能播放吗？
可以。导入时会把音频文件复制到新电脑的库目录并更新路径，所以迁移后照常播放，不依赖旧机器的文件路径。
导出能不能看进度 / 中途取消？
可以。导出时弹出进度面板，显示进度条、已处理数量、当前文件名、实时用时，并有「取消导出」按钮；取消会自动删除未完成的半成品包。完成后还会弹一条带总用时的提示，可一键打开导出文件夹。
导入时文件太多，能只导一部分吗？
可以。扫描结果页的文件列表可勾选，或用「高级过滤」按文件名包含/排除、大小范围预先筛掉；也可以先建好收藏夹，之后用「按收藏夹导出」只带走那一组。
启动后为什么要先看一段动画？
那是品牌启动动画（粒子 + 声波 + 均衡器音柱），最短展示约 1.4 秒、最长 4.5 秒兜底，期间在做数据库与界面初始化。若你用的是单文件便携版 exe，双击后还会先有一次性的自解压（约 1–2 秒黑屏），这是打包形态固有的，安装版/解压目录版无此等待。
